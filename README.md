# 简易即时通信工具

## 项目标题

SimpleChat - 基于P2P的局域网即时通信工具

## 项目简介

SimpleChat是一个基于Python开发的简易即时通信工具，采用P2P（点对点）架构，无需中心服务器即可在局域网内实现即时通信功能。主要功能包括自动发现聊天组成员、一对一消息传输、广播消息和文件传输。

### 核心特性

- 🚀 **零配置部署**：启动即用，自动发现局域网内其他用户
- 🔄 **实时通信**：支持一对一和广播消息传输
- 📁 **文件共享**：可靠的TCP文件传输，支持大文件
- 💻 **跨平台**：支持Windows、Linux、macOS系统
- 🎯 **轻量级**：纯Python实现，无需复杂依赖

## 安装指南

### 系统要求

- Python 3.6或更高版本
- 支持的操作系统：Windows 7+/Linux/macOS 10.10+
- 网络环境：局域网环境（有线或无线网络）

### 安装步骤

1. **克隆项目程序到主机**

2. **命令提示符下进入程序所在目录**，win+r，cd+空格+程序所在目录

3. **运行程序**
   
   ```bash
   python chat_tool.py
   ```

## 使用方法

### 启动程序

直接运行主程序文件：

```bash
python chat_tool.py
```

### 界面说明

程序启动后，将显示以下界面：

1. **左侧面板** - 在线成员列表
   
   - 显示当前在线的聊天组成员
   - 包含本机信息（标记为"我"）
   - 显示其他成员的用户名和IP地址

2. **右侧面板** - 聊天窗口
   
   - 上方：消息显示区域
   - 下方：消息输入和功能按钮

3. **底部面板** - 文件传输进度
   
   - 显示文件传输进度条和状态信息

### 基本操作流程

1. **自动发现成员**
   
   - 程序启动后自动发送广播消息
   - 等待其他客户端响应并显示在成员列表中

2. **发送消息**
   
   - **一对一消息**：选择成员列表中的目标用户，输入消息后点击"发送"
   - **广播消息**：输入消息后点击"广播"按钮，向所有在线成员发送

3. **发送文件**
   
   - 选择目标成员
   - 点击"发送文件"按钮
   - 选择要发送的文件
   - 接收方选择保存位置后开始传输

4. **刷新成员列表**
   
   - 点击"刷新成员"按钮重新搜索在线成员

### 快捷键

- `Enter`：发送消息
- 鼠标点击：选择聊天对象

## 功能特性

### 核心功能

- ✅ **自动成员发现**：基于UDP广播的智能发现机制
- ✅ **实时消息传输**：支持文本消息的即时发送和接收
- ✅ **消息广播**：一键向所有在线成员发送消息
- ✅ **文件传输**：基于TCP的可靠文件传输，支持进度显示
- ✅ **成员状态管理**：自动检测离线成员并更新列表
- ✅ **手动刷新**：支持手动触发成员列表刷新

### 技术特性

- **智能网络适配**：自动识别有线、无线网络环境
- **多广播支持**：兼容不同网络配置的广播发现
- **连接可靠性**：TCP/UDP混合传输模式
- **资源优化**：自动清理无效连接，减少资源占用
